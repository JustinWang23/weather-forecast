# Weather Forecast

## Install

Install taro-cli globally

`yarn global add @tarojs/cli@1.3.33 / npm install -g @tarojs/cli@1.3.33`

Install dependencies

`yarn / npm install`

## Use

Run the application and import project from WeChat Devtools

`yarn start / npm run start`
